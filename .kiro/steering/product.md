# Product Overview

KZ-Code は、AI駆動開発（AI-DLC）における基礎理論・設計原理を学ぶための日本語学習アプリケーション。
開発者が AI と協働する上で必要な知識体系を、体系的に学習できる環境を提供する。

## Core Capabilities

- **学習コンテンツ配信**: AI駆動開発の基礎理論・設計原理をインタラクティブに学べる教材提供
- **認証・ユーザー管理**: Supabase Auth によるユーザー認証（ログイン / サインアップ / パスワードリセット）
- **AI アシスタント**: Anthropic Claude API を活用した学習支援・対話型コンテンツ
- **進捗管理**: ダッシュボードによる学習進捗の可視化
- **図解・ビジュアライゼーション**: Mermaid.js による概念の図解表現

## Target Use Cases

- AI駆動開発を学びたい開発者が、基礎理論を体系的に学習する
- 設計原理や開発プロセスの概念を、図解やインタラクティブなコンテンツで理解を深める
- オンボーディングを通じてパーソナライズされた学習体験を得る

## Value Proposition

- 日本語ネイティブな学習体験（UI・コンテンツともに日本語対応）
- AI駆動開発に特化した体系的なカリキュラム
- Claude API による対話型学習と即座のフィードバック
- MDX による柔軟なコンテンツ管理
