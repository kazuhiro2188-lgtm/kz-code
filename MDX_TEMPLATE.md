# MDXコンテンツテンプレート

## 構造

```mdx
# レッスンタイトル

## はじめに

このレッスンでは、[テーマ]について学びます。

## セクション1: [サブタイトル]

[内容]

### 重要なポイント

- [ポイント1]
- [ポイント2]
- [ポイント3]

## セクション2: [サブタイトル]

[内容]

## 実践例

[実践的な例やコード]

## まとめ

- [要点1]
- [要点2]
- [要点3]
```

## 用語解説の使用

MDXコンテンツ内で用語を使用すると、自動的にアンダーラインが表示され、クリックでポップアップが表示されます。

例：
```mdx
**インフラストラクチャ**について学びます。
```

## 図解の追加

Mermaidダイアグラムを使用できます：

```mdx
\`\`\`mermaid
graph TD
    A[開始] --> B[処理]
    B --> C[終了]
\`\`\`
```

## 章末課題

章末課題は自動的に表示されます（セクションごとに1つの課題が定義されている場合）。
